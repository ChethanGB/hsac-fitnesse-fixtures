This test ensures that we can double click on buttons, based on their displayed text, and on labels, even if the associated input is invisible.

We use a mock server running to serve the HTML defined in this page.


!define HTML { {{{
<html>
<body>
<input type='text' name='set-me' id='set-me'/>
<button ondblclick='setValue()'>Double Click Me</button>
<script>
function setValue() {
    document.getElementById('set-me').value = 'Robert C. Martin'
}
</script>
</body>
</html>}}} }


|script      |mock xml server setup|
|add response|${HTML}              |
|$url=       |get mock server url  |

|script                |browser test   |
|seconds before timeout|1              |
|open                  |$url           |
|double click          |Double Click Me|
|check|value of        |id=set-me      |Robert C. Martin|

|script|mock xml server setup|
|stop                        |
